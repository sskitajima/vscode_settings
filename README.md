# vscode settings

Collection of vscode settings files to reuse the settings across PCs.

## Export

```
code --list-extensions > extensions_list.txt
```

## Import

```
PowerShell -ExecutionPolicy RemoteSigned "install_extensions.ps1" "extensions_list.txt"
```
